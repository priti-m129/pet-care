🐾 Pet Care App

Pet Care App is a web application designed to help pet owners manage their pets’ health and daily care in one place. The platform allows users to track medical records, vaccination schedules, and prescriptions, as well as manage appointments with veterinarians.

The application provides an easy-to-use dashboard where users can monitor their pet’s health history and receive reminders for important activities such as vaccinations and check-ups. It also supports features like prescription management and pet-related service information to improve overall pet care.

This project is built using React.js for the frontend, Spring Boot for the backend, and MySQL for database management, ensuring a scalable and efficient system for managing pet care information.

🚀 Key Features

Pet profile management

Vaccination tracking and reminders

Prescription management

Appointment scheduling

User dashboard for pet health records
